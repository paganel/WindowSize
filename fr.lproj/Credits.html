Qu'est-ce ? WindowSize est un menu système Mac OS X (Snow Leopard, 10.6) pour redimensionner une fenêtre aux dimensions HD standard.
Limitations Pour le moment, ne fonctionne qu'avec les applications scriptables.
Licence WindowSize est une application gratuite développée comme hobby. Toutefois, si vous trouvez qu'elle est utile, vous pouvez faire un don.
Les premiers 200 CHF de dons (annuels) seront utilisés pour couvrir les frais de nom de domaine Internet et d'hébergement. Tout montant au delà sera reversé à la Fondation Theodora .
