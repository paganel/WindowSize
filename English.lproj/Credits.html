What is it? WindowSize is a Mac OS X (Snow Leopard, 10.6) system menu for setting window sizes to HD standard dimensions.
Limitations Currently only works for scriptable applications.
License WindowSize is a free application developed as a hobby. But if you feel this is a useful application, you should make a donation.
The first 200 CHF donated (yearly) will be used to cover the Internet domain and hosting fees. Anything beyond that will be donated to the Theodora Children's Trust.
